# Energy-simulations
This is simulations that involve Battery systems and wind systems, all starting from the Rincon Mora and Min chen model, using arbitrary values.
It is a jupyter notebook with the blocks as follows:
1. Rincon Mora and Gabrial A. model using a liner function instead of the given Exponential equation.
2. Thermal lumped model of a battery system where the current value was altered to show its influence on temperature.
3. Two ODE model that looked at the core and surface temperature.
4. Two ODE system where the values of thermal Resistance was altered in order to see its effects on the disparity between core and surface and the effectr of the thermal coefficient particularly on the steady state value of the surface which was nothing the only disparity was at the difference between core and surface indicating that the rate of heat flow changed with the change of Thermal coefficient.
